This Plugin enables you to add your feed links to a widget area.

== Description ==

This Widget shows links (you can set this up) to your

* rss2 post feed
* rss2 comments feed
* atom post feed
* atom comment feed 

in your Sidebar.
You can choose if the user should see a little help link to the wiki post "What are feeds and what the hell is rss".

== Installation ==

Installation as usual.

1. Unzip and Upload all files to a sub directory in "/wp-content/plugins/".
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Go to your widget admin page and choose a place for your new Widget.
1. You like what you see?
